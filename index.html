Get started

  1. On any device (iPhone, Android, computer): open abhyaasa-manthana.web.app, choose Parent / Student or Teacher, and create an account. Abhyaasa is invite-only while in preview — new accounts are reviewed by an administrator before first use, so after signing up you'll see an “awaiting approval” screen until you're approved. On your phone you can Add to Home Screen to use it like an app.
  2. On Android (full app with offline logging + reminders): tap Download Android App above, open Abhyaasa.apk, allow “Install unknown apps” if prompted, then Install.
  3. Set up your child’s profile and start logging — you’re ready to practice! 🎵
